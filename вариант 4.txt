Доска объявлений.

В работе создается и отображается доска объявлений.

1. Создайте класс Ad (объявление). У этого класса должны быть приватные свойства head (Заголовок) , text (содержание), data (дата публикации), views (количество просмотров).

2. Создайте класс ImgAd (объявление с изображением), унаследовав его от класса Ad. Добавьте этому классу свойство img (адрес изображения).

3. Создайте класс  BoldAd (выделенное объявление), унаследовав его от класса ImgAd.

4. В каждом классе сделайте метод printAd(), который будет печатать объявление на экране. В классе Ad только заголовок (h3) и текст, В классе ImgAd  добавится изображение, в классе BoldAd - весь текст выделен полужирным и заголовок отделен линией. 

5. Создайте файл ads.json запишите в него данные 7 объявлений разных типов по образцу.
[
    {"head": "Продаю гараж", "text": " Теплый гараж недалеко от дома, цена договорная"},
    {"head": "Пианино", "text": "Производство 'Красный Октябрь', отличное состояние", "img": "piano.jpg"},
    {"head": "Пропала собака", "text": "Дворняга, на боку пятно", "img": "dog.jpg", "bold":1}
]
6. Прочитайте файл ads.json, сделайте из его данных соответствующий массив ads с помощью функции json_decode(). 

7. Сделайте из массива ads новый массив объектов objAds, в который запишите объекты соответствующих типов (Ad, ImgAd и BoldAd).
Указания для создания объетов - если свойства img нет или оно пустое создаем объект класса Ad, если есть - класс ImgAd, если есть img и установлено свойство bold - класс BoldAd.   

8. Сделайте форму для добавления объявления. Поля формы - Заголовок, Текст, Изображение.   

9. Если была отправлена форма - добавьте новое объявление в конец массива objAds 

10. Выведите все объекты на экран перебором массива objAds методом printAd().

11. Сохраните информацию из массива objAds в файл ads.json.
